Witaj w projekcie!

To jest szablon startowy dla Twoich projektów HTML/CSS/JS.

Przykładowy element

Otwórz konsolę przeglądarki (F12), aby zobaczyć przykładowy output z JavaScript.

Jak używać tego szablonu?

  1. Skopiuj folder example-template
  2. Zmień nazwę folderu na nazwę swojego projektu
  3. Edytuj pliki HTML, CSS i JS
  4. Dodaj link do nowego projektu w głównym index.html
